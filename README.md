# Trademark Multilingual Notes

A compact study app for the basics of Japanese trademark law, explained in multiple languages (English, German, Italian).

## Features

- Topic-based explanations of trademark law
- Multilingual switching (EN / DE / IT)
- Search across topics, tags, and terms
- Bookmark important topics
- Works as a Progressive Web App (PWA)

## Purpose

This app is designed as a simple “basic textbook” companion to a terminology dictionary app.  
It focuses on understanding the structure of trademark law rather than memorizing isolated terms.

## Topics included

- Purpose of trademark law
- Definition of a trademark
- Registration principle
- Distinctiveness
- Grounds for refusal (Articles 3 and 4)
- First-to-file principle
- Application to registration flow
- Term and renewal
- Scope of rights and infringement basics
- Licensing and the Madrid system

## Tech

- HTML / CSS / JavaScript
- JSON-based content
- GitHub Pages for hosting
- Service Worker for offline support

## Usage

1. Open the app in your browser
2. Select your language (EN / DE / IT)
3. Browse or search topics
4. Bookmark important entries

## Notes

This app is a study aid and does not provide legal advice.
